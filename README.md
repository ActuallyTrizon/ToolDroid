# DroidPulse 📱

**DroidPulse** is a Python-based system monitor built specifically for Termux. It provides status insights into your device's hardware, focusing on battery health and status.

## ✨ Features
* **Status Monitoring:** Tracks battery percentage and temperature.
* **Health Check:** Displays battery status (Charging, Discharging, etc.).
* **Lightweight:** Written in Python for Android.

## 🚀 Installation & Usage
1. Install Termux from GitHub or F-Droid.
2. Install dependencies:
   ```bash
   pkg update && pkg install python termux-api -y
3. Download the source code.
4. Extract the zip.
5. Run ```droidpulse.py```

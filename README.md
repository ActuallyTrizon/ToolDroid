# DroidPulse 📱

**DroidPulse** is a Python-based system monitor built specifically for Termux. It provides real-time insights into your device's hardware, focusing on optimization and battery health.

## ✨ Features
* **Real-time Monitoring:** Tracks battery percentage and temperature.
* **Health Check:** Displays battery status (Charging, Discharging, etc.).
* **Lightweight:** Written in pure Python for maximum performance on mobile.

2. Install dependencies:

   pkg update && pkg install python termux-api -y
